# Industrial-carbon-emissions-in-China
This repository contains high-resolution industrial carbon emission data for China, covering the period from 2000 to 2020. The dataset was developed using a machine learning-based framework that integrates land-use probability and geospatial datasets at a 30m resolution, offering an enhanced spatial understanding of industrial carbon emissions. 
### Industrial Carbon Emissions Data for China (2000-2020)

This dataset contains high-resolution industrial carbon emissions data for China from 2000 to 2020, generated using a machine learning-based framework. The data is provided at a 30-meter resolution and covers spatiotemporal emission patterns across the country.
